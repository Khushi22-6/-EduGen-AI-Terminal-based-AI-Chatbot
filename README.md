This project is a command-line AI-powered tutoring application called EduGen AI. It serves as an interactive, highly focused educational assistant designed to run directly inside your computer's terminal.
# 🎓 EduGen AI

EduGen AI is a simple AI learning assistant built with **Python** and the **Google Gemini API**. It helps students understand concepts, solve problems, write code, debug errors, and prepare for interviews through a simple terminal-based chat interface.

Instead of giving direct answers only, EduGen AI focuses on helping users learn and understand the topic.

---

## ✨ Features

- 📚 Answer questions from different subjects
- 💡 Explain concepts in simple language
- 💻 Write and debug code
- 📝 Generate notes and summaries
- ❓ Create quizzes and flashcards
- 🎯 Help with interview preparation
- 🚀 Suggest project ideas
- 📖 Guide students with assignments
- 💬 Easy-to-use terminal chat interface

---

## 🛠️ Built With

- Python
- Google Gemini API
- Google GenAI SDK
